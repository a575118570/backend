# 小皮 (XP) 資料庫連接設定

## 1. 小皮面板準備

1. 開啟**小皮面板**，啟動 **MySQL** 服務
2. 記下連線資訊：
   - 主機：通常 `localhost` 或 `127.0.0.1`
   - 埠：`3306`
   - 使用者：通常 `root`
   - 密碼：小皮預設常為空或 `root`

## 2. 設定 .env

編輯 `backend/.env`：

```env
# 小皮 MySQL 連線（依實際修改密碼與資料庫名）
DATABASE_URL=mysql+aiomysql://root:root@localhost:3306/anime_db
```

- 密碼為空時：`mysql+aiomysql://root:@localhost:3306/anime_db`
- 密碼為 root 時：`mysql+aiomysql://root:root@localhost:3306/anime_db`

## 3. 安裝 MySQL 驅動

```powershell
cd backend
pip install aiomysql cryptography
```

## 4. 建立資料庫（可選）

若 `anime_db` 不存在，可執行：

```powershell
python create_db.py
```

或手動在 phpMyAdmin 建立資料庫 `anime_db`。

## 5. 測試連線

```powershell
python db_connect.py
```

## 6. 啟動後端

```powershell
python main.py
```

首次啟動會自動建立資料表並匯入種子資料。
